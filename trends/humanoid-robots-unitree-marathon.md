# 휴머노이드 로봇의 약진 — Unitree 바퀴 로봇 + 휴먼노이드 하프 마라톤

> 한 줄 요약: Unitree가 롤러스케이트와 아이스링크 스케이트까지 타는 바퀴 로봇을 공개하고, 휴머노이드 로봇 하프 마라톤 대회에서 호너 로봇이 1·2·3위를 석권하며 인간보다 훨씬 빠른 속도(11m/s)를 보여줬습니다.

## 핵심 요약
- **Unitree 바퀴 로봇**: 부드러운 움직임, 롤러스케이트·아이스링크 스케이트 가능
- **휴머노이드 로봇 하프 마라톤**: 호너 로봇이 1·2·3위 모두 차지
- **인간보다 빠른 속도**: 인간들과 같이 출발해서 압도적으로 추월
- **MIRMI Bolt**: 75kg 로봇이 최대 11m/s 속도로 달림 (인간 단거리 세계기록 12.4m/s에 근접)
- **상용화 가속화**: 로봇이 사람과 같은 환경에서 함께 활동하는 시대 도래

## 본문

### Unitree 바퀴 로봇 — 부드러운 움직임의 새 차원

중국 로봇 회사 **Unitree**가 새로운 영상을 공개했습니다. 이번에는 바퀴를 단 휴머노이드 로봇입니다.

#### 가능한 동작들

| 능력 | 설명 |
|------|------|
| 매끄러운 이동 | 매우 부드러운 움직임, 자연스러운 무게 중심 이동 |
| 롤러스케이트 | 롤러스케이트를 신고 자유롭게 활주 |
| **아이스링크 스케이트** | 빙판 위에서 미끄러운 표면을 자유롭게 활주 |

#### 의미

- 바퀴 + 다리의 하이브리드 구조 → 다양한 환경 대응
- 서비스 환경(공항, 쇼핑몰)에서 활용 가능성
- 자연스러운 모션 → 사람과 같은 공간에서 어색함 없음

### 휴머노이드 로봇 하프 마라톤 대회

#### 대회 결과

| 순위 | 로봇 |
|------|------|
| 1위 | Honner 휴머노이드 로봇 |
| 2위 | Honner 휴머노이드 로봇 |
| 3위 | Honner 휴머노이드 로봇 |

**호너(Honner) 로봇이 1·2·3위를 모두 석권**했습니다.

#### 경기 모습

- **인간 선수들과 같이 출발**
- 인간들이 이제 막 뛰기 시작할 때 로봇은 이미 빠르게 가속
- 빠른 속도로 인간을 **추월**
- 압도적인 속도로 결승선 통과

#### 의미

- 휴머노이드 로봇의 **장거리 주행 능력 검증**
- 배터리 효율, 관절 내구성, 균형 유지 능력 모두 입증
- 마라톤 거리(약 21km)를 사람보다 빠르게 완주

### MIRMI Bolt — 압도적인 속도의 휴머노이드

또 다른 화제의 로봇은 **MIRMI Bolt**입니다.

| 항목 | 수치 |
|------|------|
| 로봇 무게 | 75kg |
| 최고 속도 | **11m/s** |
| 형태 | 인간형 (휴머노이드) |

#### 인간 기준과 비교

| 종목 | 속도 |
|------|------|
| 인간 단거리 세계 기록 (우사인 볼트) | 약 12.4m/s |
| **MIRMI Bolt** | **11m/s** |
| 일반인 100m 달리기 | 약 5~7m/s |

이미 **인간 세계 기록의 88%** 수준이며, 평범한 사람보다는 압도적으로 빠릅니다.

### 휴머노이드 로봇 시대의 의미

#### 환경 호환성

휴머노이드 로봇은 인간 사회의 인프라(계단, 의자, 도구)를 그대로 사용할 수 있습니다. 별도의 환경 개조 없이 도입 가능하다는 점에서 매우 매력적입니다.

#### 응용 분야

| 분야 | 활용 가능성 |
|------|-----------|
| 공장 자동화 | 인간 노동자 대체 |
| 가사 노동 | 가정에서 가사 도우미 |
| 위험 작업 | 화재, 재난 현장 투입 |
| 배송/물류 | 라스트 마일 배송 |
| 노인 돌봄 | 신체 능력 보조 |

#### 데이터 학습 — Meta의 인도 공장 사례와 연결

Meta가 인도 공장에서 노동자 데이터를 수집하는 것도 결국 이런 휴머노이드 로봇의 학습 데이터로 활용될 가능성이 큽니다.

```
인간 노동자 작업 데이터 수집 → AI 학습 → 휴머노이드 로봇 자동화
```

#### 5년 안에

- 공장 노동의 상당 부분이 휴머노이드 로봇으로 전환
- 가정용 휴머노이드 로봇 시범 출시

#### 10년 안에

- 인간 노동자의 상당수가 AI/로봇으로 대체
- 새로운 직업 생태계 재편
