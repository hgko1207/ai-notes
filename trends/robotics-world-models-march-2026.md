# 로보틱스 & 월드 모델 동향 — 서울 월드 모델, Inspirito 4D, 로봇 근황

> 한 줄 요약: 서울 거리를 AI로 재현하는 월드 모델, 영상을 3D 세계로 변환하는 Inspirito, 볼트를 조이는 로봇 손, 건담식 메카 조종까지 — 가상과 현실의 경계가 빠르게 무너지고 있습니다.

## 핵심 요약
- **서울 월드 모델** (KAIST·네이버·서울대): 네이버 스트리트뷰로 서울 거리 시뮬레이션, 재난 상황도 가상 생성
- **Inspirito**: 영상 클립을 넣으면 탐험 가능한 **3D 월드**로 변환, 오픈소스(14B) 공개
- **NVIDIA Kimodo**: 프롬프트로 3D 모션 생성, 로봇 학습 데이터에 유용
- **로봇 근황**: 링커봇 정교한 손, 건담식 메카 팔 제어 등 실용적 발전
- **포켓몬고**: 1.43억 명이 모은 **300억+ 이미지 데이터셋**으로 배송 로봇 AI 개발

## 본문

### 서울 월드 모델 — 한국에서 만든 도시 시뮬레이션

KAIST, 네이버 AI Lab, 서울대가 공동으로 **서울 월드 모델(Seoul World Model)**을 공개했습니다. 실제 대도시를 배경으로 세계를 시뮬레이션하는 AI 모델입니다.

#### 어떻게 만들었나

네이버 AI Lab이 보유한 **네이버 지도 스트리트뷰 데이터**를 활용했습니다. 네이버 지도를 만들면서 수집한 거리 데이터가 있기 때문에, 이를 기반으로 실제 서울의 지형을 반영한 영상을 AI로 생성할 수 있습니다.

#### 결과물

생성된 영상이 실제 서울을 촬영한 것과 거의 구분이 안 될 정도의 품질을 보여줍니다:

- **공덕역** 주변 → **강변북로** 진입 → **한강**, **여의도** 전경
- 실제 건물, 도로, 지형이 그대로 재현
- 차량 시점의 운전 시뮬레이션도 가능

#### 응용 가능성

| 활용 분야 | 설명 |
|-----------|------|
| **자율주행 데이터** | 실제 도로 환경의 학습 데이터 대량 생성 |
| **운전 시뮬레이션** | 가상 서울에서 운전 연습 |
| **재난 시뮬레이션** | 날씨 변경, 홍수, 토네이도 등 가상 재난 상황 생성 |
| **도시 계획** | 건물 신축, 도로 변경 시 미리 시뮬레이션 |

특히 흥미로운 것은 **재난 시뮬레이션**입니다. 서울에 홍수가 나면 어떻게 될지, 토네이도가 발생하면 어떤 모습일지를 월드 모델로 생성해볼 수 있습니다. 자율주행 AI 학습에서도 극한 상황 데이터를 만들 수 있어 매우 유용합니다.

### Inspirito — 영상을 3D 세계로 변환

Inspirito에서 공개한 **실시간 4D 세계 모델**이 큰 화제가 됐습니다. 이 회사의 슬로건이 인상적입니다:

> "우리는 영상을 제작하지 않습니다. 우리는 비디오를 통해 세상을 만들어냅니다."

#### 작동 방식

1. 아무 영상 클립을 넣습니다 (길 걸어간 영상, 생일 파티 영상, 애니메이션 등)
2. AI가 영상을 분석해서 **3D 월드**를 생성합니다
3. 생성된 월드를 **직접 탐험**할 수 있습니다

#### 활용 예시

| 입력 영상 | 결과 |
|-----------|------|
| 길 걸어간 영상 | 그 길을 자유롭게 돌아다닐 수 있는 3D 환경 |
| 생일 파티 촬영 | 파티 공간을 탐험하는 체험 |
| 애니메이션 장면 | 애니메이션 세계를 직접 걸어다니기 |

어떤 영상이든 넣으면 그 영상의 세계를 탐험하는 **게임 같은 경험**이 가능합니다.

#### 오픈소스 공개

놀랍게도 **코드와 가중치(Weight)가 모두 공개**됐습니다. 14B(140억) 파라미터 모델로, 직접 다운로드해서 돌려볼 수 있습니다.

### NVIDIA Kimodo — 프롬프트로 3D 모션 생성

NVIDIA에서 공개한 **Kimodo**는 텍스트 프롬프트만으로 3D 인간 모션을 생성하는 도구입니다.

#### 기능

- "뛰어가서 점프하는" 동작 → 프롬프트 입력 → 3D 모션 데이터 생성
- 포즈 세밀 조절 가능
- 로봇 학습용 움직임 데이터 제작에 최적

#### 실제 로봇 적용

더 놀라운 것은 이 기술이 **실제 로봇에도 적용**된다는 점입니다. 텍스트로 휴먼노이드 로봇을 실시간 제어하는 시연이 공개됐습니다.

| 텍스트 명령 | 로봇 동작 |
|-------------|-----------|
| "Clap hands" | 박수 치기 |
| "Raise both arms" | 양팔 들기 |
| "Step backward" | 뒤로 물러나기 |
| "Kick left leg" | 왼발 차기 |

시뮬레이션(Omniverse)에서 만든 동작이 실제 로봇에서도 그대로 동작한다는 것이 NVIDIA가 강조하는 핵심 포인트입니다.

### 로봇 근황 — 만화가 현실이 됩니다

#### 링커봇 링커핸드 (베이징)

베이징 링커봇의 **링커핸드**는 인간 수준의 손 동작을 구현한 로봇 손입니다.

- 손가락으로 **볼트·너트 조이기** 가능
- 형태가 다양한 부품에도 적응
- 여러 개를 한 번에 돌리는 것도 가능

정밀 제조업에서의 활용 가능성이 높습니다.

#### 건담식 메카 조종 — 현실화

만화에서 보던 거대 로봇 조종이 현실이 됐습니다.

- 사람이 팔에 장치를 장착
- 뒤에 있는 대형 로봇 팔이 1:1로 동작을 따라함
- **주먹**을 쥐면 로봇도 주먹, **따봉**을 하면 로봇도 따봉
- 에어백이 터질 정도의 힘으로 타격 가능

#### 인간형 로봇의 진화

최근 로봇들은 피부를 씌워 외형도 인간과 가깝게 만들고 있습니다. 외형뿐 아니라 기능적으로도 빠르게 발전 중입니다.

### 포켓몬고 — AI 역사상 최대 시각 데이터셋

가장 흥미로운 뉴스 중 하나입니다. 포켓몬고를 만든 **나이앤틱(Niantic)**이 놀라운 사실을 공개했습니다.

#### 숫자로 보는 규모

| 항목 | 수치 |
|------|------|
| 포켓몬고 플레이어 수 | **1억 4,300만 명** |
| 수집된 이미지 수 | **300억+ 개** |
| 데이터 유형 | 사진 + AR 스캔 데이터 |
| 규모 | AI 역사상 가장 큰 실제 시각 데이터셋 |

#### 어떻게 수집했나

포켓몬고 유저들이 게임을 하면서 카메라로 촬영한 사진, AR 스캔 데이터 등이 모두 수집되고 있었습니다. 유저들은 게임을 즐기고 있었지만, 그 이면에서 나이앤틱은 **전 세계 지형 데이터**를 구축하고 있었던 것입니다.

#### 활용 방향

이 데이터를 활용해 **배송 로봇의 시각적 내비게이션 AI**를 개발 중입니다. 실제 거리의 모습, 지형, 장애물 등을 로봇이 인식하고 이동하는 데 활용합니다.

### 피터 틸의 소 목걸이 스타트업

마지막으로 색다른 AI 적용 사례입니다. 피터 틸(Peter Thiel)이 **소 목에 두르는 AI 목걸이** 스타트업에 20억 달러를 투자했습니다.

- 소의 위치, 행동을 실시간 추적
- 앱에서 버튼을 누르면 **소가 원하는 위치로 이동**
- 기존의 카우보이가 소를 모는 방식을 AI가 대체
- **카우고리즘(Cowgorithm)**이라는 알고리즘으로 농업 생산성 향상

## 실전 활용 / 팁

1. **자율주행/로보틱스 개발자**: 서울 월드 모델로 한국 도로 환경 학습 데이터 생성 가능
2. **3D/게임 개발자**: Inspirito의 오픈소스(14B)로 영상 → 3D 월드 변환 실험
3. **모션 데이터 필요 시**: NVIDIA Kimodo로 프롬프트 기반 3D 모션 생성
4. **농업/축산 관계자**: AI 기반 가축 관리 기술 동향 주시

## 마무리

이번 주 로보틱스와 월드 모델 동향을 보면, **가상과 현실의 경계가 빠르게 무너지고 있다**는 것을 체감할 수 있습니다. 서울 거리가 AI로 재현되고, 아무 영상이든 3D 월드로 변환되며, 텍스트 명령 하나로 로봇이 움직입니다. NVIDIA가 GTC에서 강조한 "시뮬레이션에서 훈련 → 현실에서 동작"이 실제로 구현되고 있는 것입니다. 포켓몬고의 300억 이미지 데이터셋은 "AI 시대에서 데이터는 곧 자산"이라는 명제를 다시 한번 증명합니다.
