# Meta, AI 학습용 직원 마우스·키보드 입력 수집 — GDPR 위반 우려와 제로 직원 시대

> 한 줄 요약: Meta가 AI 학습 데이터 확보를 위해 직원의 마우스 움직임과 키 입력까지 수집하는 MCI 프로그램을 가동했고, 5월 20일부터 전 세계 인력 10% 감축까지 시작하며 "제로 직원 시대"라는 충격적인 미래를 시사하고 있습니다.

## 핵심 요약
- **MCI(Model Capability Initiative)**: Meta의 직원 행동 학습 프로그램, 마우스·키보드·드롭다운 선택까지 모두 수집
- **5월 20일부터 10% 인력 감축**: 전 세계 인력 감축 시작, AI 에이전트 적극 도입과 동시에 진행
- **GDPR 위반 가능성**: 화이트칼라 직원에게 배달 노동자 수준의 실시간 감시 적용 — 유럽 GDPR 위반 우려
- **인도 노동자 데이터 수집**: 인도 공장 노동자 머리에 카메라 부착 → 노동 데이터 수집 진행 중
- **노동의 자동화**: 궁극적으로 화이트칼라 + 블루칼라 모두 AI/로봇 대체 시나리오

## 본문

### Meta의 MCI 프로그램

#### 무엇인가

**MCI(Model Capability Initiative)**는 Meta의 AI 학습 데이터 확보 프로그램입니다.

#### 수집 데이터

| 데이터 | 활용 |
|------|------|
| 마우스 움직임 | 화면 조작 패턴 학습 |
| 키 입력 | 작업 흐름 학습 |
| 드롭다운 메뉴 선택 | UI 인터랙션 학습 |
| 키보드 단축키 사용 | 효율적 작업 패턴 학습 |
| 검토·개선 작업 | 품질 관리 패턴 학습 |

#### 수집 대상

**Meta 실리콘밸리 직원 전원**입니다. 즉, 세계 최고 수준의 IT 직원들이 컴퓨터로 하는 모든 작업이 학습 데이터가 되고 있습니다.

#### 의도

> AI가 아직 인간 행동을 잘 복제하지 못하는 영역의 학습 데이터 확보

특히 다음과 같은 영역이 타깃입니다.

- 컴퓨터 사용(Computer Use) AI 강화
- 복잡한 UI 조작 자동화
- 의사결정 패턴 학습

---

### 진짜 문제 — Meta의 인력 감축과 동시 진행

#### 5월 20일부터 10% 감축

| 일자 | 조치 |
|------|------|
| 2026년 5월 20일 | 전 세계 인력 10% 감축 시작 |
| 동시 | AI 에이전트 적극 도입 |

> AI에게 사람의 작업을 학습시키면서 동시에 그 사람들을 해고하는 구조

#### 흐름

```
1. 우수 직원의 컴퓨터 작업을 학습
2. AI 에이전트가 그 작업을 모방
3. 그 직원을 해고
4. AI 에이전트로 대체
5. 비용 절감 + 자동화 완성
```

#### "제로 직원 시대"의 가능성

직원의 모든 작업을 AI가 학습한다면, 결국 그 직원이 필요 없어지는 미래가 올 수 있습니다. 메타는 이 가능성을 가장 빠르게 실현하려는 회사 중 하나로 보입니다.

---

### GDPR 위반 가능성 — 화이트칼라 감시 시작

#### 유럽의 시각

> "이 조치는 화이트칼라 직원에게 기존 배달 노동자 수준의 실시간 감시를 적용하는 것이며, 유럽 GDPR 위반 가능성이 높다."

#### 무엇이 문제인가

| 영역 | 기존 노동자 | 메타 직원 (현재) |
|------|----------|----------------|
| 배달/플랫폼 노동자 | 실시간 위치, 작업 시간 추적 | - |
| 화이트칼라 | 일반적으로 자율적 | **마우스·키보드까지 추적** |

화이트칼라 직원도 이제 플랫폼 노동자 수준의 감시 대상이 되고 있습니다. 유럽 GDPR은 이런 광범위한 데이터 수집에 엄격하므로, 법적 분쟁이 예상됩니다.

---

### 노동 데이터 — 화이트칼라뿐만이 아니다

#### 인도 공장 노동자 데이터 수집

Meta는 또한 인도 공장 노동자들에게서도 데이터를 수집하고 있다고 보도됐습니다.

| 항목 | 내용 |
|------|------|
| 장소 | 인도 공장 |
| 방법 | 노동자 머리에 카메라 부착 |
| 목적 | 물리적 노동 패턴 데이터 수집 |
| 활용 | 휴머노이드 로봇 학습 데이터 |

#### 통합 시나리오

| 단계 | 내용 |
|------|------|
| 1단계 | 화이트칼라 작업 학습 → 사무직 자동화 |
| 2단계 | 블루칼라 작업 학습 → 공장 자동화 |
| 3단계 | 두 영역 통합 → 종합적 인간 노동 자동화 |

---

### 시사점 — 1인 창업의 시대

#### 노동 자동화의 가속

| 영역 | 자동화 진행 |
|------|----------|
| 화이트칼라 | Meta MCI, AI 에이전트 |
| 블루칼라 | 휴머노이드 로봇 + 인도 공장 데이터 |
| 통합 | 가까운 미래 |

#### 결론적 흐름

> "결국 1인 창업이 답인 시대가 되지 않을까?"

노동이 자동화되는 시대에 살아남으려면, **고용자가 아닌 1인 창업자(Solopreneur)**로 자리매김하는 것이 중요해질 수 있습니다.

## 실전 활용 / 팁

- **데이터 프라이버시 인식 제고**: 어떤 회사에서 일하든 본인의 작업 데이터가 어떻게 수집·활용되는지 확인하는 습관을 기르세요
- **개인 스킬 차별화**: AI가 학습할 수 없는 영역(창의성, 공감, 복잡한 협상 등)에 대한 강점을 키워두는 것이 중요합니다
- **1인 창업 준비**: 고용자가 아닌 창업자/프리랜서로 자리매김할 수 있는 스킬을 미리 준비하세요
- **GDPR/개인정보보호법 모니터링**: 본인 회사가 EU 사업장이라면 데이터 수집 정책을 검토받을 권리가 있습니다

## 마무리

Meta의 MCI 프로그램과 인력 감축 동시 진행은 AI 시대 노동 시장의 가장 어두운 시나리오를 현실로 보여주고 있습니다. 자기 작업이 AI 학습 데이터로 사용되고, 학습이 끝나면 자신이 대체된다는 구조는 디스토피아적입니다. 동시에 이는 모든 직장인에게 **"내가 가진 것이 단순한 작업 수행 능력뿐인지, 아니면 그 이상인지"**를 자문하게 만드는 신호이기도 합니다. 1인 창업, 차별화된 전문성, 인간 고유의 가치 — 이런 것들을 진지하게 고민해야 할 시점입니다.
