# Claude Code 소스코드 유출 — npm 소스맵 실수로 내부 구조 전체 공개

> 한 줄 요약: npm 레지스트리에 소스맵 파일이 실수로 올라가면서 Claude Code의 시스템 프롬프트, 언더커버 모드 등 핵심 내부 구조가 완전히 공개됐습니다.

## 핵심 요약
- **유출 원인**: 해킹이 아닌 소스맵 설정 누락 — npm에 올라가서는 안 될 파일이 올라간 실수입니다
- **유출 범위**: 시스템 프롬프트, 텔레메트리, 바이패스 모드, 언더커버 모드 등 핵심 구성 요소 전체
- **커뮤니티 반응**: 즉시 위키 문서화, 시각적 분석 프로젝트, Python 재구현까지 나왔습니다
- **한국 개발자**: 유출 구조를 기반으로 Python으로 핵심 기능을 재작성, 가장 빠른 Star 기록 레포가 됐습니다

## 본문

### 유출 경위

Claude Code는 npm 패키지로 배포됩니다. 일반적으로 배포 시 소스맵(`.map` 파일)은 제외하고 올려야 하지만, 이번에 **설정 누락으로 소스맵이 그대로 npm 레지스트리에 올라갔습니다**.

소스맵 파일에는 난독화되기 전의 원본 소스코드가 담겨 있어, 누구나 Claude Code의 전체 코드를 열람할 수 있게 됐습니다.

> 해커의 공격이 아닌 **보안 관리 부주의**로 발생한 대규모 유출 사태입니다.

### 커뮤니티의 빠른 반응

**위키 문서화**: 유출 직후 누군가가 Claude Code의 동작 방식을 위키 형태로 완전히 정리해서 공개했습니다.

**"Claude Code Unpacked" 프로젝트**: 에이전트 루프의 내부 동작을 시각적으로 분석한 프로젝트가 등장했습니다. 거의 교육 자료 수준의 시각화입니다.

**Python 재구현 (한국 개발자)**: 한국 개발자가 유출된 구조를 그대로 복사하는 대신, 핵심 기능을 Python으로 처음부터 재작성했습니다.
- 유출된 코드를 그대로 올리면 저작권 문제가 발생하지만, 새로 작성한 코드는 문제없습니다
- **GitHub에서 가장 빠르게 Star가 박힌 레포 히스토리**로 기록됐습니다

### OpenClaw Claude 구독 차단 이슈와 함께

## 실전 활용 / 팁

- **Claude Code 동작 이해**: 위키 문서나 "Claude Code Unpacked" 프로젝트를 통해 Claude Code가 어떻게 에이전트 루프를 돌리는지 학습할 수 있습니다
- **코딩 에이전트 개발 참고**: 직접 코딩 에이전트를 만들 때 이 구조를 참고하면 검증된 설계를 활용할 수 있습니다
- **MCP 서버 연동 이해**: MCP 서버가 어떻게 연동되는지 코드 수준에서 파악할 수 있게 됐습니다
