# OpenAI Sora 종료, 그리고 영상 AI의 대전환

> 한 줄 요약: OpenAI가 Sora를 6개월 만에 종료하며 AGI에 집중하겠다고 선언한 사이, Seedance 2.0·DaVinci MaQi·LTX 2.3 등 오픈소스 영상 AI가 빠르게 그 자리를 채우고 있습니다.

## 핵심 요약
- **Sora 종료**: 30일 리텐션 8% 미만으로 PMF 미달성, 디즈니 투자 철회까지 — OpenAI는 AGI·과학 연구에 집중하는 방향으로 전환합니다
- **Seedance 2.0**: 소리와 함께 영상을 생성하는 멀티모달 모델, CapCut에 통합되어 전세계 무료 체험 가능합니다
- **오픈소스 영상 AI 약진**: DaVinci MaQi Human(15B, 한국어 지원), LTX 2.3 + IDyolo(인물 반영), Prism Audio(자동 오디오) 등 도구가 쏟아지고 있습니다
- **Matrix Game 3.0**: 오픈소스 월드 모델로 720p 실시간 인터랙티브 게임을 즉석 생성합니다
- **ComfyUI 동적 VRAM**: 로컬 환경에서 VRAM 부족 문제를 해소하는 기술이 등장했습니다

---

## 본문

### 1. OpenAI Sora — 6개월 만에 종료

2025년 하반기 화려하게 등장했던 Sora가 결국 서비스를 종료합니다. 디즈니의 투자 철회와 함께, API와 ChatGPT 내 영상 생성 기능도 차례로 중단될 예정입니다.

#### 왜 실패했나 — 리텐션이 말해줍니다

a16z의 2025 소비자 AI 보고서에 따르면, 주요 AI 앱의 30일 리텐션은 다음과 같습니다.

| 서비스 | 30일 리텐션 |
|--------|------------|
| ChatGPT | **68%** |
| Gemini | **57%** |
| 주요 소비자 앱 평균 | **30%** |
| **Sora** | **8% 미만** |

토스 이승건 대표의 PMF(Product-Market Fit) 기준으로 보면 더 명확합니다:
- **20%**: 괜찮은 회사
- **40%**: 유니콘
- **70%**: 세상을 바꾸는 제품
- **Sora 8%**: PMF 미달성

#### OpenAI의 전략 변경

OpenAI는 영상 생성 대신 다음에 집중하겠다고 밝혔습니다:
- AI 모델 자체의 발전
- 과학 연구 가속
- AGI(범용 인공지능) 달성

#### 경쟁사들의 반응

Sora 종료 소식에 경쟁사들이 빠르게 반응했습니다:
- **Google Gemini**: "소라 안 되죠? 재미나이에서 만드세요"
- **일론 머스크**: "Grok 이미지 생성 능력을 두 배로 늘리겠다"

### 2. Seedance 2.0 — Sora의 가장 유력한 대안

바이트댄스의 Seedance 2.0이 정식 공개됐습니다. **Dreama AI**에서 바로 사용할 수 있습니다.

| 항목 | 내용 |
|------|------|
| 핵심 기능 | 소리와 함께 영상 생성 (멀티모달) |
| 퀄리티 | 디즈니/픽사급 애니메이션도 가능 |
| 접근성 | **CapCut에 통합** — 한국 포함 전세계 무료 체험 |
| 플랫폼 | Dreama AI |

CapCut은 이미 전세계적으로 널리 쓰이는 편집 도구인 만큼, Seedance 2.0의 접근성은 Sora와는 비교가 안 될 정도로 높습니다.

---

### 3. DaVinci MaQi Human — 오픈소스 영상 생성의 강자

완전 오픈소스로 공개된 15B 파라미터 영상 생성 모델입니다.

| 항목 | 스펙 |
|------|------|
| 파라미터 | **15B** |
| 한국어 지원 | O |
| 특징 | 음성과 동시에 영상 생성 |
| 5초 영상 생성 | H100 기준 **2초** |
| 1080p 영상 생성 | H100 기준 **38초** |
| LTX 2.3 대비 승률 | **60.9%** |
| 용량 | 61.2GB (V램 넉넉해야 합니다) |

한국어를 지원한다는 점이 특히 주목할 만합니다. 다만 61.2GB라는 용량 때문에 로컬 환경에서는 고사양 GPU가 필수입니다.

---

### 4. LTX 2.3 + IDyolo(아이디올로) — 내 얼굴을 넣은 영상

특정 인물의 **얼굴과 레퍼런스 음성**을 반영해서 영상을 생성할 수 있는 기술입니다.

- 오픈소스로 공개되어 있습니다
- **ComfyUI**에 적용 가능하여 워크플로우에 쉽게 통합됩니다
- 개인 맞춤형 영상 콘텐츠 제작에 큰 가능성을 열어줍니다

---

### 5. Prism Audio — 영상에 자동으로 오디오 입히기

오디오 없는 영상에 자연스러운 오디오를 자동으로 생성해주는 모델입니다.

| 항목 | Prism Audio | MM Audio (기존) |
|------|------------|-----------------|
| 파라미터 | **500M** | 1.03B |
| 성능 | **더 우수** | 기준 |

파라미터 수가 절반도 안 되면서 성능은 더 좋다는 점이 인상적입니다. 영상 AI와 결합하면 "프롬프트 하나로 소리 있는 영상"이 완성되는 파이프라인을 구축할 수 있습니다.

---

### 6. ComfyUI 동적 VRAM — 로컬 환경의 구원자

ComfyUI에 **Dynamic VRAM** 기술이 도입됐습니다.

- 시스템 RAM 사용량이 감소합니다
- 기존에 VRAM 부족으로 오류가 나던 문제가 해결됩니다
- 모델 생성 시간도 단축됩니다

로컬에서 영상 AI를 돌리는 분들에게 반가운 소식입니다. GPU 업그레이드 없이도 더 무거운 모델을 돌릴 수 있게 된 것입니다.

---

### 7. Matrix Game 3.0 — 게임을 즉석에서 만드는 월드 모델

오픈소스 **월드 모델**로, 돌아다니면서 상호작용 가능한 게임을 실시간으로 생성합니다.

| 항목 | 스펙 |
|------|------|
| 해상도 | **720p** 실시간 |
| 모델 구조 | **28B MoE** |
| VRAM 요구 | **56GB 이상** |
| 특징 | 장편 비디오 생성, 인터랙티브 |

아직은 VRAM 요구 사항이 높아 일반 사용자가 직접 돌리기는 어렵지만, 게임 개발의 미래를 엿볼 수 있는 기술입니다.

---

### 8. Image2Live2D — 이미지 한 장으로 버츄버 제작

애니메이션 캐릭터 이미지 **한 장**만으로 버츄버(VTuber)용 레이어를 자동 분리해주는 기술입니다.

- 눈, 입, 머리카락 등을 별도 레이어로 분리합니다
- Live2D 리깅에 바로 활용할 수 있습니다
- 오픈소스 버전이 곧 출시될 예정입니다

버츄버를 시작하고 싶지만 일러스트 + 리깅 비용이 부담이었던 분들에게 게임 체인저가 될 수 있습니다.

## 실전 활용 / 팁

1. **지금 당장 영상 AI를 체험하고 싶다면**: CapCut에서 Seedance 2.0을 무료로 사용해 보세요. 별도 설치 없이 바로 가능합니다
2. **로컬 환경에서 영상 생성을 하고 싶다면**: ComfyUI + LTX 2.3 + IDyolo 조합을 추천합니다. Dynamic VRAM 기능도 함께 활용하세요
3. **한국어 프롬프트를 쓰고 싶다면**: DaVinci MaQi Human이 한국어를 지원합니다. 다만 61.2GB 용량과 높은 VRAM이 필요합니다
4. **영상에 오디오를 자동으로 붙이고 싶다면**: Prism Audio를 영상 생성 파이프라인 뒤에 연결하면 됩니다
5. **버츄버를 저비용으로 시작하고 싶다면**: Image2Live2D 오픈소스 버전 출시를 주목하세요

## 마무리

Sora의 종료는 단순한 하나의 서비스 실패가 아닙니다. "중앙집중형 유료 영상 AI"보다 "오픈소스 + 멀티모달 + 로컬 환경"이 더 빠르게 진화하고 있다는 시장의 신호입니다. Seedance 2.0은 CapCut을 통해 접근성을, DaVinci MaQi와 LTX 2.3은 오픈소스로 자유도를, ComfyUI는 로컬 환경의 효율을 각각 높이고 있습니다. 영상 AI의 주도권이 거대 기업에서 커뮤니티와 오픈소스로 빠르게 이동하고 있는 흐름을 주목해야 합니다.
