# Concepts

AI 핵심 개념과 기술 원리를 정리합니다.

## 다루는 내용

- **아키텍처** — Transformer, Attention, Diffusion 등 모델 구조
- **학습 기법** — Fine-tuning, RLHF, DPO, LoRA 등
- **활용 패턴** — RAG, Agent, Function Calling, MCP 등
- **평가/안전** — Alignment, Hallucination, Red Teaming, Benchmark
- **인프라** — Tokenization, Embedding, Vector DB, Inference 최적화

## 목적

트렌드와 도구는 빠르게 변하지만 개념은 오래 유효합니다.
기술의 원리를 이해하면 새로운 도구가 나와도 빠르게 적응할 수 있습니다.
