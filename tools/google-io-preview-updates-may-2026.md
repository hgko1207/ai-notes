# 구글 I/O 사전 공개 묶음 — Gemini 3.1 Flash Lite, 새 이미지 모델, Stitch, Formuli, 웹훅

> 한 줄 요약: 구글이 5월 19일 Google I/O를 앞두고 Gemini 3.1 Flash Lite 출시, 새 이미지 모델 테스트, Stitch 디자인 개선, Formuli 마케팅 AI, Gemini API 웹훅 등 굵직한 업데이트를 연달아 공개했습니다.

## 핵심 요약
- **Gemini 3.1 Flash Lite**: 100만 토큰당 입력 $0.25, 매우 저렴한 빠른 모델
- **딥마인드 신규 이미지 모델**: GPT Image 2 수준 테스트 중, I/O에서 공개 가능성
- **Stitch 업데이트**: 네이티브 이미지 생성 대폭 업그레이드, I/O를 위한 "Something Big" 준비
- **Formuli 카탈로그**: 제품 URL만 등록하면 브랜드 맞춤 캠페인 이미지 자동 생성
- **Gemini API 웹훅**: 긴 작업 완료 시 자동 호출, 에이전트 워크플로우 편의 개선

## 본문

### Gemini 3.1 Flash Lite — 가성비 모델

가볍고 빠르면서도 저렴한 모델이 정식 출시됐습니다.

#### 가격

| 항목 | 가격 (100만 토큰당) |
|------|---------------------|
| 입력 | **$0.25** |
| 오디오 입력 | $0.5 |
| 출력 | $1.5 |

#### 활용 시나리오

| 시나리오 | 장점 |
|--------|------|
| 대량 데이터 처리 | 압도적으로 저렴 |
| 빠른 응답 필요 | Flash 시리즈 특성 |
| 단순 작업 자동화 | 비용 대비 충분한 성능 |
| 챗봇/콜센터 | 응답 속도와 비용 균형 |

#### Gemma 4와의 경쟁?

| 모델 | 특징 |
|------|------|
| Gemini 3.1 Flash Lite | 유료 API, 빠르고 저렴 |
| Gemma 4 (오픈소스 + MTP) | 무료 로컬 실행, 3배 가속 |

Gemma 4 MTP가 너무 강력해지면 Flash Lite의 시장 경쟁력을 위협할 수 있어, 구글이 Gemma 4 MTP를 일시적으로 숨긴 의혹의 배경이 됩니다.

### Formuli 카탈로그 — 마케팅 자동화 AI

#### 어떤 도구인가

구글 랩스의 마케팅용 AI 제품 **Formuli**에 **카탈로그** 기능이 추가됐습니다.

#### 작동 방식

```
1. 제품 페이지 URL 입력
2. AI가 제품 정보 자동 추출
3. 브랜드 맞춤 캠페인 이미지 자동 생성
```

#### 활용 시나리오

| 용도 | 결과물 |
|------|------|
| 스토리 광고 | 스토리에 바로 쓸 수 있는 이미지 |
| 인스타그램 피드 | 브랜드 톤에 맞춘 게시물 이미지 |
| 페이스북 광고 | 캠페인용 카로셀 이미지 |
| 4×4 그리드 광고 | 한 번에 16개 변형 광고 |

#### 누가 좋아할까

- **이커머스 운영자**: 매번 디자이너 의뢰 없이 광고 이미지 생성
- **소상공인**: 마케팅 비용 절감
- **광고 대행사**: 빠른 시안 제작 가능

### Gemini API 웹훅 — 에이전트 작업 편의성 향상

#### 어떤 기능인가

AI 작업은 오래 걸리는 경우가 많습니다(예: Deep Research, 긴 코드 생성). **웹훅(Webhook)**을 통해 작업 완료 시 지정된 URL로 자동 호출됩니다.

#### 작동 흐름

```
1. API 호출 시 웹훅 URL 지정
2. AI 작업 처리 (긴 시간)
3. 작업 완료
4. 웹훅 URL로 결과 자동 전송
5. 후속 작업 자동 실행
```

#### 에이전트 워크플로우 활용

| 단계 | 예시 |
|------|------|
| 1단계 | Deep Research 호출 (10분 소요) |
| 2단계 | 결과를 받은 즉시 → Code 생성 호출 (10분 소요) |
| 3단계 | 코드 완성 → 자동 배포 호출 |
| 4단계 | 배포 완료 → 슬랙 알림 |

별도의 폴링(주기적 확인) 없이도 **에이전트 체인**을 효율적으로 구축할 수 있습니다.
