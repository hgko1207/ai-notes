# 클로드 코드 2026년 3월 하순 업데이트 — Auto 모드, Computer Use, 그리고 새로운 도구들

> 한 줄 요약: Claude Code에 Auto 모드와 Computer Use가 추가되고, Impeccable·Expect·Understudy 같은 서드파티 도구들이 등장하면서 AI 코딩을 넘어 AI 데스크톱 자동화 시대가 본격화되고 있습니다.

## 핵심 요약
- **Auto 모드**: 위험한 작업만 자동으로 차단하면서 나머지는 알아서 승인해주는 안전한 자동 실행 모드입니다
- **Computer Use**: 마우스·키보드·화면을 직접 제어해서 브라우저, 스프레드시트, PPT 등을 조작할 수 있습니다
- **iMessage 플러그인**: 아이폰 문자로 Claude와 대화할 수 있는 공식 플러그인이 출시되었습니다
- **Impeccable / Expect / Understudy**: 디자인 품질 향상, 브라우저 UI 테스트, 시연 기반 자동화 등 Claude 생태계를 확장하는 도구들이 등장했습니다

## 본문

### Auto 모드 — 안전하게 자동 승인

기존에도 `dangerously skip` 옵션으로 모든 승인을 건너뛸 수 있었지만, 이름 그대로 "위험한" 방식이었습니다. **Auto 모드**는 이 문제를 해결한 안전한 버전입니다.

#### 작동 방식
1. `claude --auto-mode`로 실행합니다
2. 일반적인 작업은 Claude가 알아서 승인하고 진행합니다
3. D드라이브 삭제, DB 날리기 같은 **위험한 작업은 자동으로 감지**하여 사용자에게 리뷰를 요청합니다

#### 사용 조건

| 항목 | 내용 |
|------|------|
| 상태 | 리서치 프리뷰 |
| 사용 가능 플랜 | 팀 플랜, 엔터프라이즈, API |
| 일반 플랜 | 아직 미지원 (토큰 소모가 크기 때문) |

토큰을 많이 소모하는 특성 때문에 일반 플랜에서는 아직 제한되어 있습니다. 향후 확대가 기대됩니다.

---

### Computer Use — AI가 마우스와 키보드를 직접 제어

이전에는 Open Claw 같은 서드파티 도구에서만 가능했던 컴퓨터 직접 제어가 이제 **Claude에서 네이티브로 지원**됩니다.

#### 할 수 있는 것들
- 브라우저 네비게이트 (웹 검색, 페이지 이동)
- 스프레드시트 조작
- PPT를 PDF로 변환
- 캘린더에 일정 첨부
- **133개 이미지를 1200x1200으로 리사이징 + 워터마크 삽입** — 딸깍 한 번으로 완성

마우스, 키보드, 화면을 직접 제어하는 방식이며, Claude Code Desktop 및 Co-Work와 연동됩니다.

#### 플랫폼 지원 현황

| 플랫폼 | 상태 |
|--------|------|
| macOS | 출시 완료 |
| Windows | 수주 내 지원 예정 |

아직 초기 단계라 동작 속도가 조금 느린 편이지만, 반복적인 단순 작업을 한 번에 자동화할 수 있다는 점에서 활용 가치가 높습니다.

---

### iMessage 플러그인 — 문자로 Claude와 대화

기존에 디스코드, 텔레그램 연동(Channels)이 있었다면, 이번에는 **아이폰 iMessage**로 Claude와 대화할 수 있는 공식 플러그인이 추가되었습니다.

별도 앱을 열지 않아도 문자 앱에서 바로 Claude에게 질문하고 답변을 받을 수 있어서, 아이폰 사용자에게 특히 편리합니다.

---

### Impeccable — AI 디자인 품질을 전문가 수준으로

Claude가 코드는 잘 짜지만 **디자인은 "AI가 만들었네" 느낌**이 나는 문제를 해결하는 AI 하네스(harness)입니다.

#### 7개 전문 분야

| 분야 | 설명 |
|------|------|
| 타이포그래피 | 폰트 선택, 글자 크기, 행간 등 |
| 색상 대비 | 접근성을 고려한 컬러 조합 |
| 공간 디자인 | 여백, 레이아웃 밸런스 |
| 모션 디자인 | 애니메이션, 전환 효과 |
| 상호작용 디자인 | 버튼, 입력 등 인터랙션 |
| 반응형 | 모바일/태블릿/데스크톱 대응 |
| colorize | 밋밋한 디자인에 색감을 입혀주는 기능 |

기존에 AI가 생성한 UI가 투박하다고 느꼈다면, Impeccable을 적용하는 것만으로 **전문 디자이너가 만든 것 같은 깔끔한 결과물**을 얻을 수 있습니다.

---

### Expect — 에이전트가 직접 브라우저에서 테스트

코드를 작성한 뒤 **실제 브라우저에서 UI를 테스트**해주는 도구입니다.

#### 작동 흐름
1. CLI로 Expect를 실행합니다
2. Playwright 기반 브라우저가 자동으로 열립니다
3. 에이전트가 테스트 계획을 수립합니다
4. 브라우저에서 실제로 클릭, 입력, 탐색을 수행합니다
5. **Pass/Fail 리포트**를 생성합니다

수동으로 브라우저를 열어 테스트하던 과정을 자동화할 수 있어서, 특히 프론트엔드 개발 시 생산성이 크게 향상됩니다.

---

### Understudy — 한 번 보여주면 알아서 반복

사용자가 **작업을 한 번 시연해주면 의도와 절차를 학습**해서 이후 자동으로 반복 수행하는 데스크톱 에이전트입니다.

#### 사용 방법
1. `teach` 명령으로 시연 녹화를 시작합니다
2. 사용자가 원하는 작업을 직접 수행합니다
3. Understudy가 단순 동작이 아닌 **의도를 기반으로 학습**합니다
4. 결과물은 **SKILL.md** 파일로 저장되어 재사용할 수 있습니다

현재는 macOS 중심으로 동작합니다. 매크로와 달리 의도를 이해하기 때문에, 화면 레이아웃이 약간 바뀌어도 유연하게 대응할 수 있다는 점이 강점입니다.

## 실전 활용 / 팁

1. **반복 작업 자동화**: Computer Use로 이미지 일괄 처리, 파일 변환 같은 노가다를 한 번에 해결할 수 있습니다
2. **프론트엔드 개발**: Expect를 도입하면 매번 수동 테스트하던 시간을 절약할 수 있습니다
3. **디자인 개선**: Claude로 UI를 만들 때 Impeccable을 함께 사용하면 퀄리티가 확 올라갑니다
4. **업무 자동화**: Understudy로 반복적인 데스크톱 작업을 시연 한 번으로 자동화할 수 있습니다
5. **Auto 모드 주의**: 토큰 소모가 크므로 팀/엔터프라이즈 플랜에서 사용하되, 위험 작업 감지를 과신하지 말고 중요한 작업은 직접 확인하는 것이 좋습니다

## 마무리

이번 업데이트의 핵심 키워드는 **"코딩을 넘어선 자동화"**입니다. Auto 모드로 승인 과정을 간소화하고, Computer Use로 마우스·키보드까지 제어하며, Understudy로 시연 기반 학습까지 가능해졌습니다. 여기에 Impeccable과 Expect 같은 서드파티 도구들이 Claude 생태계를 빠르게 확장하고 있습니다. AI 코딩 도구가 이제 진짜 "AI 비서"로 진화하고 있다는 걸 체감할 수 있는 업데이트입니다.
