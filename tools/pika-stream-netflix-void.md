# Pika Stream 1.0 + Netflix VOID — AI 화상 미팅과 영상 물체 삭제

> 한 줄 요약: Pika의 실시간 비디오 에이전트가 화상 미팅에 AI로 참여하는 기능을 열었고, Netflix 연구진의 VOID 기술은 영상에서 물체를 물리 법칙까지 반영해서 삭제합니다.

## 핵심 요약
- **Pika Stream 1.0**: 미팅 링크를 전송하면 AI가 화상 채팅에 직접 참여하는 실시간 비디오 에이전트입니다
- **Netflix VOID**: 영상에서 원하는 물체를 제거할 때 해당 물체와의 상호작용까지 자동으로 반영합니다
- **VOID 오픈소스**: Netflix의 VOID 기술은 오픈소스로 공개됐습니다
- **영상 편집의 새로운 차원**: 단순한 인페인팅을 넘어 물리적 상호작용까지 처리하는 수준입니다

## 본문

### Pika Stream 1.0 — AI가 화상 미팅에 참여

Pika Labs가 **Pika Stream 1.0**을 출시했습니다. 이 기능의 핵심은 AI와 **화상 채팅**이 가능하다는 것입니다.

#### 작동 방식

1. "Hey, can you join this meeting?" 메시지와 함께 미팅 링크를 전송합니다
2. AI가 화상 미팅에 직접 참여(join)합니다
3. 실시간으로 AI와 영상 통화가 가능합니다

기존의 텍스트 기반 AI 대화나 음성 대화를 넘어, 이제는 **화면에 직접 등장하는 AI**와 대화하는 시대가 열렸습니다. 가상 인터뷰 연습, 외국어 말하기 연습, 가상 고객 서비스 등 다양한 용도로 활용 가능합니다.

---

### Netflix VOID — 영상에서 물체를 지우면 물리까지 바꿔줍니다

Netflix 연구진이 발표한 **VOID(Video Object and Interaction Deletion)** 기술입니다.

#### 일반 영상 편집과의 차이

| 기능 | 일반 인페인팅 | VOID |
|------|-------------|------|
| 물체 제거 | 가능 | 가능 |
| 배경 복원 | 가능 | 가능 |
| 물리적 상호작용 반영 | **불가** | **가능** |

#### 실제 사례들

- **무게 있는 물체 제거**: 소파에 앉아 베개가 눌린 영상에서 사람을 제거하면, 베개가 다시 원래 모양으로 돌아옵니다
- **물리 연쇄 반응**: 영상 중간에 있는 물체를 제거하면, 그 물체가 없었다면 발생했을 물리 반응으로 자동 수정됩니다
- **볼링 사례**: 볼링공이 핀을 쓰러뜨리는 영상에서 볼링공을 제거하면 핀이 쓰러지지 않는 영상으로 바뀝니다
- **고양이 방해 제거**: 고양이가 작업을 방해하는 영상에서 고양이를 제거하면 방해가 없는 자연스러운 영상이 됩니다

#### 오픈소스로 공개

VOID는 오픈소스로 공개되어 있어, 영상 편집 파이프라인에 직접 통합할 수 있습니다.

---

### OpenCut — 캡컷의 오픈소스 대안

이 시기에 **OpenCut**도 공개됐습니다. 틱톡의 CapCut(캡컷)과 유사한 기능의 오픈소스 영상 편집 툴입니다.

- 다운로드해서 로컬에서 사용 가능합니다
- 프리미어 프로 같은 고가 편집 소프트웨어의 대안이 됩니다

## 실전 활용 / 팁

- **Pika Stream**: 외국어 말하기 연습, 인터뷰 준비, 가상 미팅 시나리오 등에 활용해볼 수 있습니다
- **VOID 오픈소스**: ComfyUI나 영상 AI 파이프라인에 VOID를 연결하면 물리 반영 영상 편집이 가능합니다
- **OpenCut**: CapCut을 대체하고 싶은 분들에게 좋은 선택입니다. 특히 편집 데이터를 외부 서버에 보내지 않고 로컬에서 처리할 수 있습니다

## 마무리

Pika Stream 1.0은 AI 인터랙션의 차원을 텍스트·음성에서 영상으로 넓혔습니다. Netflix VOID는 영상 편집의 수준을 한 단계 끌어올린 기술로, 오픈소스로 공개된 덕분에 누구나 활용할 수 있습니다. 영상 AI 분야에서의 혁신 속도가 점점 빨라지고 있습니다.
