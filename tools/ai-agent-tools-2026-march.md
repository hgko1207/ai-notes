# AI 에이전트 도구 총정리 — 서브 에이전트부터 AI 직원 회사까지

> 한 줄 요약: Codex 서브 에이전트, Claude Router, Browser Use CLI, Liva까지 — AI 에이전트가 실제로 일하고, 비용을 아끼고, 회사를 운영하는 시대의 도구들을 정리합니다.

## 핵심 요약
- **Codex 서브 에이전트**: 여러 에이전트를 동시에 굴려서 작업 속도 대폭 향상 (토큰은 많이 소모)
- **Claude Router**: 질문 난이도별로 최적 모델에 라우팅, API 비용 **78% 절감**
- **Browser Use CLI 2.0**: 터미널에서 브라우저를 자동 제어하는 CLI 도구
- **Liva**: AI 에이전트가 은행 계좌, 법인, 이메일까지 갖고 자율적으로 회사를 운영

## 본문

### Codex 서브 에이전트 — 병렬 작업의 시대

OpenAI의 Codex에서 **서브 에이전트(Sub-Agent)** 기능이 공식 지원되기 시작했습니다. 하나의 메인 에이전트가 여러 개의 하위 에이전트를 소환해 동시에 작업을 시키고, 결과를 통합하는 방식입니다.

#### 왜 서브 에이전트가 필요한가

기존에는 하나의 에이전트가 모든 작업을 순차적으로 처리했습니다. 문제는:

| 문제 | 설명 |
|------|------|
| 컨텍스트 한계 | 작업이 많아지면 컨텍스트가 일찍 꽉 참 |
| 속도 | 하나씩 순서대로 처리하니 느림 |
| 전문성 | 하나의 에이전트가 모든 영역에 다 능숙하긴 어려움 |

서브 에이전트는 이 문제를 해결합니다:
- 각 에이전트가 **독립적인 컨텍스트**를 가짐 → 낭비 없음
- **병렬 실행** → 3개 에이전트가 동시에 일하면 3배 빠름
- 역할 분담 → API 디자이너, 백엔드 개발자 등 전문화 가능

#### 사용 팁

> **중요**: 프롬프트에 "서브 에이전트"라는 단어를 명시적으로 넣어야 호출됩니다. 안 넣으면 기본적으로 서브 에이전트를 사용하지 않습니다.

#### 서브 에이전트 컬렉션 (GitHub)

GitHub에 "Codex 서브 에이전트 컬렉션" 레포가 화제가 됐습니다. 다양한 전문 서브 에이전트가 정의돼 있습니다:
- API 디자이너
- 백엔드 개발자
- 프론트엔드 개발자
- 데이터베이스 설계자
- 기타 등등

이 에이전트들을 전부 불러서 동시에 작업시키면, 풀스택 서비스를 한 번에 만드는 것도 가능합니다.

> 토큰 사용량은 확 늘어나지만, 작업량과 속도 면에서 큰 이점이 있습니다.

### Claude Router — API 비용 78% 절감

Open Claw 같은 에이전트 프레임워크의 가장 큰 단점은 **API 비용**입니다. Claude Router는 이 문제를 정면으로 해결합니다.

#### 작동 원리

모든 요청이 들어오면 **1밀리초 이내에 난이도를 분석**하고, 가장 효율적인 모델로 라우팅합니다.

| 질문 유형 | 라우팅 모델 | 비용 |
|-----------|------------|------|
| 2 + 2 = ? | DeepSeek | 최저가 |
| 기사 요약해줘 | 중간급 모델 | 중간 |
| React 컴포넌트 만들어줘 | Claude Sonnet | 높음 |
| 복잡한 아키텍처 설계 | Claude Opus | 최고가 |

핵심은 **모든 질문에 비싼 모델을 쓸 필요가 없다**는 것입니다. 간단한 질문에는 저렴한 모델을, 복잡한 질문에만 고급 모델을 사용하면 전체 비용이 크게 줄어듭니다.

- **절감율**: 최대 78%
- **라이선스**: 100% 오픈소스
- **지연시간**: 라우팅 판단에 1ms 미만

### Browser Use CLI 2.0 — 터미널에서 브라우저 자동화

MCP(Model Context Protocol)가 화제인 요즘이지만, **CLI 기반 도구**도 주목받고 있습니다. Browser Use CLI는 터미널에서 브라우저를 자동 제어하는 도구입니다.

#### 왜 CLI인가

Claude Code, Codex 등 주요 AI 코딩 도구들이 모두 **터미널 기반**으로 동작합니다. MCP로 복잡하게 연결하지 않아도, CLI 명령어로 바로 브라우저를 조작할 수 있습니다.

#### 사용 예시

```
브라우저 유즈 설치 → 내 브라우저랑 연결해 → X.com으로 이동해
```

이렇게 지시하면 브라우저가 알아서 접속하고, "최근 알림 뭐야?"라고 물으면 정보를 가져와줍니다.

#### CDP 연동

Chrome DevTools Protocol(CDP)과 연결해서 개발자 도구의 데이터를 직접 활용할 수도 있습니다. 웹 스크래핑, 테스트 자동화, 데이터 수집 등에 유용합니다.

### Liva — AI 자율 직원 플랫폼

가장 미래적인 도구입니다. AI 에이전트들이 **실제 법인, 은행 계좌, 이메일, 전화번호**까지 소유하면서 자율적으로 회사를 운영합니다.

#### 각 AI 에이전트가 갖는 것

- 개인 소유 컴퓨터
- 개인 소유 은행 계좌
- 개인 소유 법인
- 개인 소유 이메일 주소
- 개인 소유 계정 정보
- 개인 소유 휴대전화

#### 수행 업무

| 업무 | 설명 |
|------|------|
| SEO/GEO | 검색 엔진 최적화 |
| 랜딩 페이지 | 웹사이트 제작 |
| 아웃바운드 영업 | 링크드인 메시지 발송 등 |
| 풀스택 앱 | 애플리케이션 개발 |
| 링크드인 포스팅 | 콘텐츠 마케팅 |
| 회계 | 재무 관리 |

사람이 전혀 개입하지 않고 AI 에이전트들끼리 협업해서 매출을 올리는 구조입니다.

#### 부가: Cloud 3D Office

Open Claw가 무슨 작업을 하고 있는지 **3D 가상 사무실** 형태로 시각화해주는 도구도 화제입니다. 게임처럼 AI 에이전트들의 업무 현황을 직관적으로 볼 수 있습니다.

### Open Claw + Premiere Pro — 영상 편집 자동화

Open Claw를 이용해서 AI로 영상을 생성하고, Premiere Pro로 가져와 편집하는 것까지 자동화한 사례가 공개됐습니다. 스킬을 잘 설정해두면(편집 스타일, 선호 효과 등) 영상 편집도 AI에게 맡길 수 있는 시대입니다.

1. **Codex 서브 에이전트**: 프롬프트에 "서브 에이전트를 사용해서" 같은 문구를 명시적으로 넣을 것

3. **Browser Use CLI**: Claude Code에서 웹 작업이 필요할 때 MCP 대신 CLI로 간편하게 연동
