# Claude Computer Use Windows 지원 + Claude Code 내 Codex 플러그인

> 한 줄 요약: Claude의 컴퓨터 직접 제어 기능(Computer Use)이 Windows까지 확장됐고, Claude Code 안에서 ChatGPT의 Codex를 사용할 수 있는 공식 플러그인도 등장했습니다.

## 핵심 요약
- **Computer Use Windows 출시**: macOS에서만 되던 마우스/키보드 제어가 Windows에서도 공식 지원됩니다
- **사무 업무 자동화**: Windows 사용자들도 반복적인 파일 작업, 스프레드시트 조작 등을 자동화할 수 있습니다
- **Codex 플러그인**: Claude Code 안에서 ChatGPT 구독을 통해 Codex를 사용할 수 있는 플러그인이 출시됐습니다
- **도구 통합 시대**: 단일 에이전트 안에서 여러 AI 모델을 선택해서 쓰는 흐름이 가속화되고 있습니다

## 본문

### Computer Use Windows 지원

이전까지 Claude의 Computer Use(컴퓨터 직접 제어) 기능은 macOS에서만 동작했습니다. 이번 업데이트로 **Windows에서도 공식 지원**됩니다.

#### 할 수 있는 것들

- 브라우저 자동 탐색
- 스프레드시트 데이터 입력 및 조작
- 파일 일괄 처리 (이미지 리사이징, 워터마크 삽입 등)
- PPT → PDF 변환
- 캘린더에 일정 및 첨부파일 추가

#### 왜 Windows 지원이 중요한가

국내 사무 환경에서는 Windows가 압도적으로 많이 사용됩니다. macOS에서만 된다는 점이 Computer Use 확산의 가장 큰 걸림돌이었는데, 이번 확장으로 훨씬 많은 사람들이 활용할 수 있게 됐습니다.

> 현재는 초기 단계라 동작 속도가 느린 편입니다. 하지만 반복적인 단순 작업에는 충분히 활용 가치가 있습니다.

### Claude Code 내 Codex 플러그인

OpenAI에 근무하는 개발자가 **Claude Code 안에서 Codex를 실행할 수 있는 플러그인**을 만들어 공개했습니다.

#### 작동 방식

1. Claude Code에 Codex 플러그인 설치
2. ChatGPT 구독 계정 연동
3. Claude Code 안에서 Codex를 서브 에이전트로 호출

```
# 기존에도 CLI 명령어로 Codex를 사용할 수는 있었지만,
# 이제 공식 플러그인으로 더 편리하게 통합됩니다
```

#### 의미

Claude Code를 사용하면서 특정 작업에는 Codex가 더 적합한 경우 플러그인으로 바로 전환할 수 있습니다. 단일 코딩 에이전트 안에서 여러 AI 모델을 목적에 맞게 활용하는 **멀티 에이전트 + 멀티 모델** 시대가 열리고 있습니다.

## 실전 활용 / 팁

- **Computer Use 시작**: Claude Desktop 또는 Claude Code에서 Computer Use 설정을 활성화한 뒤, 반복적인 파일 처리 작업부터 시작해보세요
- **속도 문제 우회**: 현재 속도가 느리므로, 50개 이상의 반복 작업처럼 시간이 오래 걸리는 일에 활용하면 체감 효과가 큽니다
- **Codex 플러그인**: Claude Code 사용자이면서 ChatGPT 구독도 있다면, 플러그인을 설치해서 두 모델을 비교해볼 수 있습니다

## 마무리

Computer Use의 Windows 지원은 국내 사무 환경에서 AI 자동화를 현실화하는 중요한 업데이트입니다. Codex 플러그인은 AI 도구 생태계가 점점 더 개방적으로 통합되고 있다는 신호입니다. 앞으로는 "어떤 AI 도구를 쓸까"보다 "어떤 작업에 어떤 모델을 조합할까"가 더 중요해질 것입니다.
