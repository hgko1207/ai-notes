# LLM Wiki + Awesome Design MD + K-Skills — 에이전트 시대의 지식 관리 리소스

> 한 줄 요약: 안드레이 카파시의 LLM Wiki 개념, 유명 기업 디자인을 역설계한 Awesome Design MD, 한국인 맞춤 MCP K-Skills까지 — 에이전트 시대에 꼭 알아야 할 개발 리소스 3종 세트입니다.

## 핵심 요약
- **LLM Wiki**: RAG 대신 LLM이 직접 위키를 작성·관리해 지식을 영속적으로 축적하는 방식입니다
- **Awesome Design MD**: 애플·스포티파이·에어비앤비 등의 디자인 시스템을 역설계한 마크다운 파일, MIT 라이선스로 무료 공개됩니다
- **K-Skills**: KTX 예매, 서울 지하철, 로또 당첨 확인 등 한국인을 위한 MCP 스킬 모음입니다

## 본문

### LLM Wiki — RAG를 대체하는 새로운 지식 관리 방식

AI 분야의 거장 **안드레이 카파시(Andrej Karpathy)**가 개인 지식 저장소 구축에 LLM Wiki 방식을 활발하게 사용하고 있다고 밝혔습니다. 관련 가이드 파일도 공개했습니다.

#### 기존 RAG vs LLM Wiki

| 구분 | RAG | LLM Wiki |
|------|-----|----------|
| 방식 | 외부 문서를 검색해 LLM에 참고 제공 | LLM이 직접 위키 문서를 작성하고 축적 |
| 특징 | 실시간 검색 가능, 최신 정보 반영 | 시간이 지날수록 지식이 풍부해짐 |
| 적합한 용도 | 방대한 외부 자료 검색 | 개인/팀 내부 지식 축적 |

#### 어디에 적용할 수 있나

- 개인 건강 기록 및 목표 관리
- 독서 노트 및 연구 아카이브
- 팀 내부 위키 구축
- 프로젝트별 지식 축적

에이전트가 작업하면서 자동으로 위키를 업데이트하게 만들면, 시간이 지날수록 더 풍부한 지식 베이스가 형성됩니다.

---

### Awesome Design MD — 대기업 디자인 시스템을 내 에이전트에게

**Awesome Design MD**는 디자인을 잘하는 유명 기업들의 디자인 시스템을 역설계하여 마크다운 파일로 정리한 프로젝트입니다.

#### 포함된 기업 디자인 시스템

- **Apple**: 애플 특유의 미니멀리즘 디자인 원칙
- **Spotify**: 음악 스트리밍 서비스 특유의 다크 테마와 컬러 시스템
- **Airbnb**: 여행 서비스에 맞는 따뜻하고 신뢰감 있는 디자인
- 그 외 다수의 디자인 선진 기업

#### 내용

각 기업 디자인 파일에는 다음 항목들이 상세히 정리되어 있습니다:
- 비주얼 테마와 전반적인 분위기
- 컬러 팔레트 (정확한 HEX 코드)
- 타이포그래피 (폰트, 크기, 줄 간격)
- 간격 및 레이아웃 원칙
- 컴포넌트별 디자인 가이드

#### 라이선스

**MIT 라이선스**로 완전 무료 공개됩니다. 상업적 프로젝트에도 자유롭게 사용할 수 있습니다.

#### 활용 방법

```
# 예시: Claude Code에 애플 디자인 적용
프롬프트: "apple-design.md 파일을 참고해서 이 UI를 
애플 스타일로 리디자인해줘"
```

### K-Skills — 한국인을 위한 MCP 스킬 모음

**K-Skills**는 한국 서비스들을 MCP(Model Context Protocol)로 연결할 수 있는 스킬 모음입니다.

#### 포함된 스킬들

| 카테고리 | 스킬 |
|----------|------|
| 교통 | KTX 예매, SRT 예매, 서울 지하철 도착 정보 |
| 금융 | 토스 증권 조회, KB증권 |
| 법률 | 한국 법령 검색 |
| 스포츠 | K리그 정보 |
| 기타 | 로또 당첨 확인, 카카오톡 Mac CLI |

한국에 특화된 서비스를 AI 에이전트와 연동할 때 K-Skills를 활용하면 빠르게 구현할 수 있습니다.

## 실전 활용 / 팁

- **에이전트에게 디자인 가이드 주기**: Claude Code나 Cursor로 UI 개발 시, Awesome Design MD에서 원하는 스타일의 파일을 컨텍스트에 포함시키면 훨씬 완성도 높은 결과물이 나옵니다
- **K-Skills 한국 서비스 연동**: 한국 특화 서비스가 필요한 에이전트를 만들 때 K-Skills 레포를 참고하면 처음부터 만들지 않아도 됩니다
- **LLM Wiki 도입**: 반복되는 프로젝트나 개인 지식 관리에 RAG 대신 LLM Wiki 방식을 도입해보면 점진적으로 더 풍부한 지식 베이스가 만들어집니다

## 마무리

세 가지 리소스 모두 에이전트 시대에 개발 생산성을 크게 높여줄 수 있는 도구들입니다. 특히 Awesome Design MD는 AI가 생성하는 UI의 품질을 한 단계 끌어올릴 수 있는 실용적인 자료이며, K-Skills는 한국 개발자들에게 더욱 유용한 에이전트 구축을 가능하게 합니다.
