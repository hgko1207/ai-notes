## 다루는 내용

- **모델 스펙** — 파라미터, 컨텍스트 윈도우, 가격 등 핵심 사양
- **모델 비교** — 용도별 모델 선택 가이드 (코딩, 글쓰기, 분석 등)
- **릴리즈 노트** — 주요 모델 업데이트 및 변경사항
- **벤치마크** — 성능 비교, 강점/약점 분석
