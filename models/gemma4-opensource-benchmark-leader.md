# Gemma 4 — 파라미터 대비 최강 성능의 오픈소스 모델 (Apache 2.0)

> 한 줄 요약: 구글이 공개한 Gemma 4는 31B 파라미터로 685B급 모델을 넘어서는 벤치마크 성능을 보여주며, Apache 2.0 라이선스로 상업적 사용까지 완전히 자유로운 오픈소스 모델입니다.

## 핵심 요약
- **Apache 2.0 라이선스**: 이전 Gemma 시리즈의 제한적 라이선스를 벗어나 상업적 사용이 완전히 자유롭습니다
- **31B 파라미터로 Arena Elo 1452점**: 685B, 397B, 754B급 모델들과 비슷하거나 더 높은 성능입니다
- **오픈소스 모델 중 3위**: 조 단위 파라미터 모델들과 경쟁하는 31B 모델의 등장입니다
- **모바일용 라인업 존재**: 4E2B 모델이 아이폰 17 Pro에서 40 tokens/s로 동작합니다
- **코드포스 2150점**: 코딩 성능도 최상위 수준입니다

## 본문

### 모델 라인업

| 모델 | 파라미터 | 특징 |
|------|----------|------|
| Gemma 4 31B | 31B | 최고 성능, 오프라인 프런티어급 지능 |
| Gemma 4 27B (MoE) | 27B (활성 파라미터 적음) | 경량 고속, MoE 구조 |
| Gemma 4 E4B | ~4B (엣지) | 에이전트 기반, 이미지/오디오 지원 |
| Gemma 4 E2B | ~2B (모바일) | 스마트폰 온디바이스, 오디오 지원 |

> **주의**: 31B/27B 모델은 오디오 인식이 지원되지 않습니다. 오디오가 필요하다면 E4B/E2B를 선택해야 합니다.

---

### 벤치마크 성능

Arena Elo 기준 비교입니다.

| 모델 | 파라미터 | Arena Elo |
|------|----------|-----------|
| Gemma 4 31B | **31B** | **1452점** |
| GPT-4o | 120B | 31B보다 낮음 |
| 경쟁 모델 A | 754B | 비슷한 수준 |
| 경쟁 모델 B | 1100B | 비슷한 수준 |
| 경쟁 모델 C | 685B | Gemma 4에 열세 |
| Qwen 3.5 | 397B | Gemma 4 31B에 열세 |

오픈소스 모델 랭킹에서 **3위**를 기록했으며, 뒤에 있는 Qwen 3.5(397B)마저 31B짜리 Gemma 4가 이겼습니다.

---

### 주요 기능

- **고급 추론**: 복잡한 멀티스텝 추론 지원
- **에이전트 기반 워크플로우**: 도구 사용, 계획 실행 가능
- **코드 생성**: 코드포스 2150점 수준의 코딩 능력
- **멀티모달**: 이미지 및 오디오 지원 (모델 버전 따라 다름)

---

### 스마트폰에서도 돌아갑니다

아이폰 17 Pro에서 Gemma 4 E2B 모델이 **40 tokens/second** 속도로 동작합니다. 무거운 작업이 아니라면 이제 스마트폰 안에서 완전 온디바이스 AI 처리가 가능합니다.

### 활용 방법

**Google AI Gallery**: 공식 데모 페이지에서 바로 테스트해 볼 수 있습니다.

**Claude Code 연동**: Claude Code에서 Gemma 4를 직접 연결해서 사용할 수 있습니다. 유료 구독 없이 전기세만으로 코딩이 가능합니다.

**다양한 파생 버전**:
- MLX 버전 (애플 실리콘 최적화)
- 양자화 버전 (VRAM 절약)
- 언세이프 버전 (구글 필터 제거, 오픈소스 특성)

## 실전 활용 / 팁

- **로컬 LLM 입문**: 31B 모델부터 시작하면 성능과 용량 균형이 좋습니다. Ollama나 LM Studio에서 바로 다운로드할 수 있습니다
- **Claude Code 비용 절감**: Claude Code의 모델을 Gemma 4로 교체하면 API 비용 없이 코딩 작업을 진행할 수 있습니다
- **상업 서비스 통합**: Apache 2.0이므로 별도 계약 없이 제품에 내장하거나 API로 서비스할 수 있습니다
- **모바일 앱 개발**: E2B/E4B 모델은 온디바이스 AI 앱 개발에 최적화되어 있습니다

## 마무리

Gemma 4는 단순히 "작은 오픈소스 모델"이 아닙니다. 31B라는 비교적 가벼운 용량으로 수백억~조 단위 파라미터 모델과 경쟁하는 파격적인 가성비를 보여줬습니다. 특히 Apache 2.0 라이선스를 채택한 점은 개인 개발자, 스타트업, 기업 모두에게 큰 의미가 있습니다. 로컬 환경에서 고성능 AI를 무료로 사용하는 시대가 본격적으로 열리고 있습니다.
